ASL Translator: American Sign Language Translate App
============================================================================================

Presiquites
-----------------
Python 3.10  
Scikit-learn 1.2.0  
Flask  
MediaPipe  

Installation
-----------------

1. Download repo.
2. Install presiquites
```
pip install scikit-learn==1.2.0
```
```
pip install flask
```
```
pip install mediapipe
```
3. Run the application
```
python app.py
```
